# How_To_Install

## Using NPM

```jsx
import { gsap } from "gsap";  
import { ScrollTrigger } from "gsap/ScrollTrigger";
```

```jsx
gsap.registerPlugin(ScrollTrigger);  
```

## Using CDN

```jsx
<script src="https://cdn.jsdelivr.net/npm/gsap@3.12.5/dist/gsap.min.js"></script>
```
